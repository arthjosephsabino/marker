Jquery Marker

The jquery marker plugin provides easy highlighting with mouse dragging.

Features

  • A big feature of Jquery Marker is to ignore HTML tags.
  • This is still a trial version and development is ongoing. Please tell me if you have a bug.
  • Css and js are customizable.
  • Color change is prototype and instability.
  • There is a possibility of other script conflicts since refreshing whole html.
  • We do not recommend setting it to <body> tag.
  • p.s. All the plugins optionally supported are MIT licensed, so as safe to use as the Marker plugin itself.

Tutorial

  • 1. Click 'create' button.
  • 2. Drag to set mark.
  • 3. Left click to change color.
  • 4. Right click to remove mark.

Download & Install

  • jQuery v1.9.0 or higher required.
  • jQueryUI v1.9.0 or higher required.
  • IE 9+ or higher required.
  • Sourcecode on Github: https://daesuni.github.io/marker/

Quick start

Here's a short code fragment, demonstrating how to use Marker at it's most basic level.

There are many more options to tailor it to your needs.

<div class="classname">
    ...
</div>

<script>
    // simple
    $('.classname').marker();

    // option
    $('.classname').marker({
        debugging : false,
        minimum : 3,
        maximum : 1500,
        colorPicker : true,
        style : {
            'background' : '#232323',
            'font-weight' : 'bold'
        },
        overlap : true
    });
</script>

Documentation

Methods

  • .marker(options)

    Create one or more marker or access an existing marker.

  • .marker('clear')

    Deletes the element of the set mark.

  • .marker('data')

    Get the order of appearance, style, and unique key of the set mark elements in the array.

  • .marker('destroy')

    Destroys the marker widget and unbind all.

Options

  • debugging (boolean, default: false)

    Set the debug mode with the console log.

  • minimum (integer, default: 3)

    Length of minimum setting mark.

  • maximum (integer, default: 1500)

    Length of maximum setting mark.

    If there is no limit, freezing may occur. We recommend 1500.

  • colorPicker (boolean, default: "true")

    You can change the color using spectrum.js.

  • style (object, default: "{ 'background' : '#232323' }")

    Apply custom style with css syntax.

  • overlap (boolean, default: true)

    Whether mark overlap is allowed or not.

EXTENSIONS / DEPENDENCIES

marker is designed to take advantage of a number of separate Javascript libraries to supply additional features. The download comes with these files included. If you deploy your code, you may wish to use or exclude these files.

spectrum.js

https://github.com/bgrins/spectrum

The spectrum.js is an option necessary to change the color of the block. It is not necessary to configure with only a single color.
